# copilot-testing

### How to Run the Game

1. Build it:
   ```
   wasm-pack build --target web
   ```
2. Install dependencies (if any):
```
npm install
```

3. Start the local server:
```
npm start
```

4. Open your browser and navigate to:
```
http://localhost:3000
```

## How to Play
- Use left/right arrow keys to move the paddle
- Press spacebar to launch the ball
